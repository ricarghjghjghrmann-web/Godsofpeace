Game


A Brutal Mystic Open-World RPG

Since May 2020, I’ve been creating a game with a clear vision: An unforgiving, atmospheric open-world RPG inspired by classic games.

This is not a game that holds your hand.

The Vision

Imagine a world inspired by Gothic, Souls-like games, and Moonstone.

  • Dark atmosphere
  • Immersive world
  • Brutal combat

You are not a hero. You start weak, lost, and replaceable. Everything must be earned.

A World That Feels Real

  • No empty spaces
  • Dense world design
  • Exploration matters

There is no minimap telling you where to go. Exploration is real again.

Combat & Atmosphere

Combat is brutal, direct, and grounded. Every hit matters.

This is not a game made by a committee. This is a raw, uncompromising vision.
